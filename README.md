# Веб-приложение для обнаружения нарушителей с использованием компьютерного зрения

Веб-приложение для обнаружения людей на охраняемом объекте в реальном времени с использованием компьютерного зрения.  
Система принимает видеопоток с камеры, выполняет детекцию людей, сохраняет события в базе данных и отправляет уведомления через Telegram.

---

## Возможности

- Прием и обработка видеопотока с веб-камеры
- Обнаружение людей на кадрах с использованием модели YOLOv5
- Сохранение скриншотов при обнаружении человека
- Хранение данных о событиях (время, камера, изображение) в базе PostgreSQL
- Отправка уведомлений в Telegram
- Веб-интерфейс для управления Telegram-чатами
- Контейнеризация приложения и базы данных с помощью Docker и Docker Compose

## Архитектура приложения

1. Видеопоток поступает с камеры.
2. Кадры обрабатываются моделью YOLOv5.
3. При обнаружении человека:
   - сохраняется скриншот;
   - событие записывается в PostgreSQL;
   - отправляется уведомление в Telegram.
4. Управление уведомлениями осуществляется через веб-интерфейс.
